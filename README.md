# RAFT-HazeToken

This repository contains a prototype implementation of a haze-aware optical flow model based on RAFT.

Modifications:
- Frozen OpenCLIP image encoder
- Global Haze Token generation
- FiLM conditioning of context features
- Research prototype for VLM-guided optical flow under haze

This implementation is part of the Hazy-VLM-Optical-Flow research project.
